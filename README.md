# 动漫图片搜索网站

一个现代化的动漫图片搜索网站，支持关键词搜索、分类筛选和多种视图模式。

## 🌟 功能特点

- 🔍 **智能搜索** - 支持关键词搜索动漫角色和场景
- 📱 **响应式设计** - 完美适配桌面端和移动端
- 🎨 **现代化UI** - 深色主题配合动漫风格配色
- 🏷️ **分类筛选** - 按角色、场景、同人、官方等分类
- 📊 **排序功能** - 按最新、热门、相关度排序
- 👀 **视图切换** - 网格和列表视图模式

## 🚀 快速部署

### 方法一：Vercel（推荐）
1. 访问 [vercel.com](https://vercel.com)
2. 注册账号并连接GitHub
3. 导入此项目仓库
4. 自动部署并获得域名

### 方法二：Netlify
1. 访问 [netlify.com](https://netlify.com)
2. 拖拽项目文件夹到网站
3. 自动部署并获得域名

### 方法三：GitHub Pages
1. 创建GitHub仓库
2. 上传所有文件
3. 在仓库设置中启用GitHub Pages

### 方法四：传统虚拟主机
1. 购买域名和主机空间
2. 通过FTP上传文件
3. 配置域名解析

## 🎯 使用说明

1. **搜索功能** - 在搜索框输入动漫角色名或关键词
2. **分类浏览** - 点击分类卡片快速筛选
3. **快速标签** - 点击热门标签进行搜索
4. **视图切换** - 在网格和列表视图间切换
5. **排序功能** - 按不同方式排序搜索结果

## 🔧 本地开发

```bash
# 启动本地服务器
python -m http.server 8000

# 或使用Node.js
npx serve .
```

访问 `http://localhost:8000` 查看网站
